# BookStore

App de ventas de libros.

## Authors

- [@gonzaloCanut](https://github.com/glcanut)

## Run Locally

Clone the project

```bash
  git clone https://github.com/glcanut/react-app.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

## Tech Stack

**Client:** React, Vite, React Boostrap

**Server:** Firebase
